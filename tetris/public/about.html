우리의 미션

테뚜리뚜(Te-ttu-ri-ttu)는 누구나 무료로, 설치 없이 바로 즐길 수 있는 웹 기반 퍼즐 게임 서비스입니다. 우리는 "장벽 없는 즐거움"이라는 철학 아래, 모든 사용자가 기기나 환경에 관계없이 동일한 게임 경험을 누릴 수 있도록 최선을 다하고 있습니다.

1984년 알렉세이 파지노프에 의해 탄생한 테트리스의 정신을 계승하면서도, 현대적인 웹 기술을 활용하여 더욱 세련되고 빠른 게임 플레이를 제공합니다.

핵심 기능

  • ✅ 실시간 전세계 랭킹 시스템 - Firebase 기반 초저지연 스코어 서버
  • ✅ 크로스 플랫폼 지원 - PC, 태블릿, 모바일 모든 환경에서 플레이 가능
  • ✅ 부드러운 60FPS 애니메이션 - 경량화된 렌더링 엔진
  • ✅ 터치 컨트롤러 지원 - 모바일에서도 쾌적한 조작감
  • ✅ 100% 무료 - 결제 없이 모든 기능 이용 가능
